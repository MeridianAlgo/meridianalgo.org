# MeridianAlgo Website & Documentation

This repository contains:
- The main MeridianAlgo website (React + Vite + Tailwind CSS)
- The official documentation site (Docusaurus, in `meridianalgo-docs/`)

## Getting Started

**For local development, you should run both the main website and the documentation site in parallel. The Documentation link in the navbar will open the local docs site (http://localhost:3000) in a new tab.**
